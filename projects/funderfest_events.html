FunderFest Events

Project Overview

Created as part of the She Codes Plus program, 2023/2024. This project proposal was to create a crowdfunding platform. Using my provious experience in the events industry, I decided to create a platform for event organisers who needed funding to get their events off the ground.

This project had two aspects, the Django Rest Framework API back end and a React JS front end. I utilised the Tailwind framework to style the front end.
